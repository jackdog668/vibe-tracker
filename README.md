# Vibe Tracker

Vibe Tracker is a local-first daily mood and productivity tracker built as a vanilla HTML, CSS, and JavaScript demo. It stores everything in `localStorage`, requires no build step, and can be deployed directly to GitHub Pages or Netlify.

## Features

- Daily check-in with vibe score, productivity score, and optional journal note
- Current and longest streak tracking with milestone feedback
- Weekly dashboard with two canvas bar charts and summary stats
- Expandable history list with delete actions
- JSON export, JSON import, and full data reset controls

## Run locally

Open `index.html` in a browser.

## Deploy

Because the project is fully static, you can deploy it to GitHub Pages, Netlify, or any static host without configuration.
